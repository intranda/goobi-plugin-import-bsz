

Dateinamenskonvention

Für identifizierte Binnenstrukturen innerhalb des Buches sind folgende Markierungen vorgesehen:

volume:	Band
text:	Text
image:	ungezählte Seiten aller Art, z.B. Abbildungen, Faltkarten, Widmung, Vita,
	Lebenslauf, leere Seiten etc.
cover_front:	Vorderdeckel, -umschlag
cover_back:	Rückdeckel, -umschlag
title_page:	Titelblatt
contents:	Inhaltsverzeichnis
preface:	Vorwort
index:	Register, Autoren- und Sachregister, Abbildungsverzeichnis
additional:	Anhang, Beilage, Berichtigungen, Errata, Corrigenda


Eindeutige Identifizierung des Titels:

Kennung = PPN Nr. aus SWB Verbund (Im Ausdruck der Katalogaufnahme in Kategorie 001 enthalten)


Bezeichnung der laufenden Scannummer: S + vierstellige laufende Nummer


Zahlentypen:

X = ohne Pagnierung
A = arabische Ziffer
R = römische Zahl in Großbuchstaben
r = römische Zahl in Kleinbuchstaben

Schema: Zahlentyp + vierstellige laufende Nummer

Zusammengefasst wird folgende Dateinamenskonvention festgelegt:
<Kennung>-S<fortlaufende Scannummer>-<Binnenstruktur>-<Zahlentyp><Seite>.tif
Beispiel:

PPN: 001815385
              Titel: Die Konstanzer Ratslisten des Mittelalters /
                     hrsg. von d. Badischen Historischen Kommission.
                     Bearb. von Konrad Beyerle
          Beteiligt: Beyerle, Konrad [1872-1933]
      Körperschaft: Badische Historische Kommission
         Erschienen: Heidelberg : Winter, 1898
             Umfang: VII, 252 S.
